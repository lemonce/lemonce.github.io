Create in menu bar

You can create new test cases from the toolbar or use the shortcut key , in which case you need to create a path to save.


Create in case editor tool bar

You can also create new test cases by click the button of case-editor, in which case you need to create a path to save.


Create in work space

The workspace is connected to local file system. Any changes on the cases such as create, rename, or remove will be synchronized to the local folder.
