页面内容：
  1. #TIMES
  2. #AUTOWAIT
  3. #LIMIT

#TIMES

决定了执行测试用例的次数，添加在主过程之前，默认值为1。

// 这个测试用例会执行十次
#TIMES 10
process main{

}


#AUTOWAIT

每条语句执行间隔时间，默认等待500毫秒。

// 将主动等待时间改为一秒
#AUTOWAIT 1000
process main{

}


#LIMIT

每条语句等待选择器元素出现时长的上限，默认值为5秒

// 将被动等待时间改为十秒
#LIMIT 10000
process main{

}
