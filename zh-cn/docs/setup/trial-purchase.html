License界面

第一次打开灵萌测软件时，会自动弹出 license 界面，如下图所示。

  • Machine Code 用户用于运行灵萌测的设备的机器码。
  • 复制机器码，用于购买License。
  • Please enter activation code 在输入框中输入激活码。
  • 点击以激活软件。
  • 点击以进入购买页面。
  • 试用客户端，点击后自动生成一段激活码到输入框中。
  • 退出license界面，点击该按钮之后，客户端也会关闭。


工具试用

用户可以先对客户端进行试用，按照以下两个步骤来操作。

点击 按钮，客户端会在激活码输入框内自动生成一段激活码。

点击 按钮，license界面即自动关闭，激活完成！

激活之后，您可以开始使用软件开展自动化测试，试用期为一个月。在试用期内打开客户端，license界面不会再次弹出。 如果您在试用中有任何的疑问，欢迎您联系我们。


工具购买

在一个月试用期结束之后，用户打开客户端，license 界面会再次弹出。再次点击 生成的激活码此时是无法进行激活的，在license界面里会出现 verified failed。

如果用户需要继续使用灵萌测，请点击 ，弹出的浏览器窗口会自动跳转到购买界面的 版块。

用户注册完成之后，去到注册的邮箱点击激活链接。

登录之后，去顶栏的 界面购买 license，购买完成之后，再回到 进入到下图的页面。共分为三个板块。

  • 1 Bind

    用户可将需要绑定的机器码复制到 Machine Code 输入框，点击 进行绑定，该机器即可永久使用 Lemonce。

  • 2 Bind List

    Bind List 中显示的是授权绑定机器的时间，机器码以及对应机器的激活码。将激活码复制到客户端弹出的 license 界面中的 Activation Code 输入框点击 即可永久激活。

    同样的，当用户更换机器时，可以将授权许可与机器解绑，点击右侧的 ，解绑的许可可以重新绑定新的机器。

  • 3 Product

    包含了产品的版本，总共的许可，已绑定的许可以及未绑定的许可。（图中产品大版本为2，总许可数为10个，已绑定1个，未绑定9个）。
